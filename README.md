# AI-Driven Behavioral Severity, Media Narrative & People Perception Modeling System

## 📌 Project Overview
This project is an AI-powered research platform designed to study how behavioral factors, language patterns, media narratives, and public perception interact during high-impact social events. The system combines Machine Learning, Natural Language Processing, simulation models, and Explainable AI to support ethical research on early intervention and social risk analysis.

---

## 🎯 Problem Statement
Public perception often escalates faster than verified facts during violent or high-profile incidents. This project aims to analyze behavioral severity, trigger language, media framing, and public perception to better understand escalation patterns and support responsible early-intervention research.

---

## 📂 Features
- 🧠 Behavioral Severity Prediction
